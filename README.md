# יומן משימות — PWA

## מה כולל החבילה

```
yoman-pwa/
├── index.html      ← האפליקציה הראשית
├── sw.js           ← Service Worker (אופליין + התראות ברקע)
├── manifest.json   ← הגדרות PWA (שם, צבעים, אייקונים)
└── icons/          ← אייקונים בכל הגדלים
    ├── icon-72.png
    ├── icon-96.png

    ├── icon-144.png
    ├── icon-152.png
    ├── icon-192.png

    └── icon-512.png
```

## איך להעלות לאינטרנט (חינמי)

### אפשרות 1: GitHub Pages (הכי קל)
1. צור חשבון ב-github.com
2. צור repository חדש (שם כלשהו)
3. העלה את כל הקבצים
4. Settings → Pages → Source: main branch
5. תקבל כתובת: https://שם-משתמש.github.io/שם-repository

### אפשרות 2: Netlify Drop (הכי מהיר — גרור ושחרר)
1. כנס ל-netlify.com
2. גרור את תיקיית yoman-pwa לדף
3. תקבל כתובת מיידית

### אפשרות 3: Vercel
1. vercel.com → New Project
2. העלה תיקייה
3. Deploy

## איך להתקין על הטלפון

### Android (Chrome) — הכי מלא
1. פתח את הכתובת ב-Chrome
2. תופיע הודעה "הוסף למסך הבית"
3. לחץ התקן — מוכן!
4. **התראות ברקע**: עובדות גם כשהמסך כבוי

### iOS (Safari)
1. פתח ב-Safari (דווקא Safari, לא Chrome)
2. לחץ על כפתור Share (ריבוע עם חץ)
3. "הוסף למסך הבית"
4. **הגבלה ב-iOS**: התראות ברקע דורשות iOS 16.4+

## יכולות האפליקציה

✅ **עבודה אופליין** — Service Worker שומר את כל הקבצים
✅ **IndexedDB** — אחסון מקומי מהיר ויציב (לא localStorage בלבד)
✅ **התראות ברקע** — גם כשהאפליקציה סגורה (Android)
✅ **Periodic Background Sync** — בדיקת תזכורות כל דקה
✅ **ייצוא/ייבוא** — גיבוי נתונים כ-JSON
✅ **Snooze** — דחיית תזכורת ב-10 דקות
✅ **כפתורי פעולה בהתראה** — "בוצע" ו"דחה" ישירות מההתראה
✅ **עדכון אוטומטי** — הודעה כשיש גרסה חדשה
✅ **אינדיקטור אופליין** — פס אדום בראש כשאין חיבור
✅ **Safe areas** — תמיכה בנוטש של iPhone
✅ **App shortcuts** — קיצור דרך ל"הוסף משימה"
